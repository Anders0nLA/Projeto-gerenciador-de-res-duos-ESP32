# Projeto-gerenciador-de-res-duos-ESP32
Sistema IoT para monitoramento em tempo real de níveis de resíduos, desenvolvido com ESP32 e sensor ultrassônico. O projeto visa otimizar a coleta seletiva através de tecnologia acessível, fornecendo dados precisos sobre o enchimento das lixeiras e permitindo gestão mais eficiente dos recursos urbanos.
